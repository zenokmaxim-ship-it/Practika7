## Описание проекта

Данный проект разработан на языке **Python** и демонстрирует работу системы резервного питания сервера с использованием **Tkinter**, **ООП** и **Pygame**.

Программа моделирует переключение питания сервера с внешней электросети на аккумуляторную батарею, а также отображает движение электрического тока и разряд аккумулятора.

---

# Используемые технологии

- Python 3.x
- Tkinter
- Pygame
- Объектно-ориентированное программирование (ООП)

---

# Реализованные задания

## Задача №1 — GUI (20 баллов)

Реализован графический интерфейс пользователя.

Возможности:

- окно программы с двумя вкладками;
- кнопка **«Отключить внешнее питание»**;
- флажок **«Генератор исправен»**.

---

## Задача №2 — ООП (25 баллов)

Создан класс **PowerSystem**.

На экране отображаются:

- сервер;
- блок внешнего питания (розетка);
- аккумуляторная батарея;
- соединительные провода.

Класс отвечает за хранение текущего состояния системы питания и управление переключением источников энергии.

---

## Задача №3 — Pygame (25 баллов)

Реализована анимация движения электрического тока.

Особенности:

- при работе от сети ток движется от розетки к серверу;
- после отключения внешнего питания движение переключается на линию батареи;
- анимация непрерывная.

---

## Задача №4 — Комплекс (30 баллов)

Реализована работа аккумулятора.

Функции:

- отображается уровень заряда батареи;
- полный заряд расходуется за 10 секунд;
- если аккумулятор полностью разрядился и генератор неисправен, сервер отключается;
- экран становится черным и выполнение программы завершается.

# Запуск проекта

1. Установить Python 3.

2. Установить библиотеку Pygame:

```bash
pip install pygame
```

3. Запустить программу:

```bash
python main.py
```

---

# Демонстрация работы

Во время работы программы можно:

- отключить внешнее питание;
- наблюдать переключение питания на батарею;
- следить за движением электрического тока;
- наблюдать уменьшение заряда аккумулятора;
- включать или отключать генератор;
- увидеть отключение сервера при полном разряде батареи и неисправном генераторе.
